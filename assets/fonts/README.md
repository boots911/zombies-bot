# Card fonts (optional)

Drop font files here to override the default system sans-serif:

| File          | Used for                          |
|---------------|-----------------------------------|
| `heading.ttf` | usernames, map names, big numbers |
| `body.ttf`    | labels, stat values               |

`.otf` also works (`heading.otf` / `body.otf`). Either file may be omitted —
whatever's missing falls back to sans-serif, so nothing breaks.

Suggestions:
- **Heading:** a clean blocky/Minecraft-style face (e.g. Minecraftia, or a
  geometric display font) for personality.
- **Body:** a legible UI sans (e.g. Inter, Roboto) for the dense numbers.
